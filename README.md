## 🛠️ How to Install Ad-Astra on Windows

Follow these steps one by one to get Ad-Astra working in Minecraft. No programming knowledge is needed.

### Step 1: Install Minecraft Java Edition

If you do not have Minecraft Java Edition:

- Visit https://www.minecraft.net/en-us/download
- Download and install the Java Edition for Windows.
- Create a Minecraft account if you don’t have one.
- Run Minecraft once to complete its setup.

### Step 2: Install Minecraft Forge

Minecraft Forge is a tool needed to run mods like Ad-Astra.

1. Open your web browser and go to:  
   https://files.minecraftforge.net/

2. Under “Minecraft Version,” select the version that matches your Minecraft (for example, 1.19).

3. Download the “Installer” file for that version.

4. Run the installer by double-clicking it.  
   Select “Install client” and click OK.

5. Wait for it to finish. This adds the mod support to Minecraft.

6. Open the Minecraft launcher.  
   Choose the Forge profile from the version list.  
   Click Play to start Minecraft with Forge.

### Step 3: Add the Ad-Astra Mod File

1. Press `Windows + R` to open the Run box.

2. Type `%appdata%\.minecraft\mods` and press Enter.  
   (If the “mods” folder does not exist, create it.)

3. Copy or move the downloaded Ad-Astra `.jar` file into the “mods” folder.

4. Close the folder.

### Step 4: Run Minecraft with Ad-Astra

1. Open the Minecraft launcher.

2. Select the Forge profile again.

3. Click Play.

4. When the game loads, you should see Ad-Astra as a loaded mod on the main menu or mods list.

You can now start a new game or load your world to try the new space tech features.
